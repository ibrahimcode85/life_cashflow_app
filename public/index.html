LIFE ACTUARIAL CASHFLOW MODEL
WELCOME!

This application, powered by Python, is designed to assist life actuaries in projecting their insurance cashflows.

It seamlessly integrates with your existing pricing Excel models. Users simply need to 'link' this application to their pricing model by specifying the directory path of the Excel file and the names of the tables and input parameters defined in the Excel fileâ€”then you're all set.

With this 'link' design, users no longer need to prepare separate input files to run the Python-based model.

Quick Start
  • Navigate to Input Settings and specify the directory path to the Excel file, along with the names of the tables and input parameters defined in the Excel file.
  • Navigate to Output Setting to specify the format and type of output to be generated.
  • Click the Run Button. The run log can be viewed directly within this application or can be generated as one of the output files for audit purposes.
